Which login should I use?
— API token works for newer inverters: MIN, MOD, MID, SPH, SPA series. Generate it in ShinePhone → Me → username → API Token. Unlimited lookback.
— Username + password works for older inverters: MIC, MAX, MAC series (including MIC 3000TL-X). Reliable up to ~30 days — data beyond that may be inaccurate.
— Not sure? Try token first. If it returns no data, switch to username + password.

Eloverblik — consumption data optional — enables real self-use savings
Eloverblik is Energinet's official smart meter data hub. Connecting it gives the dashboard your real hourly household consumption, making self-use and savings figures accurate instead of estimated.

Get your token: Log in to eloverblik.dk with MitID → click the person icon (top right) → Datadeling → Opret Token → copy it here. Token is read-only and stays in your browser only.
Note: eloverblik data has a 1-day delay — yesterday's consumption is not yet available. Leave blank to skip.
