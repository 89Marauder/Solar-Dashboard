# Solar Dashboard

A personal solar export revenue dashboard for Growatt inverters with live Nordpool spot prices.

## Features
- 15-minute MTU resolution matching production to Nordpool DK2 prices
- NRGi deduction calculations (4 øre/kWh handling + 0.5 øre/kWh Energinet tariff)
- Gross profit per MTU
- Insights panel with best/worst MTUs, peak hours, and daily summaries
- CSV export

## Usage
Open the dashboard URL, paste your Growatt API token in Setup, select a date range, and hit Fetch.
